О проекте

DAMP - это современная платформа для управления криптовалютными активами, разработанная как курсовой проект.

Система предоставляет инструменты для безопасного хранения, анализа и управления цифровыми активами.

Возможности

  • Управление портфелем активов
  • Анализ рыночных данных
  • Персонализированные рекомендации
  • Безопасное хранение
  • Интеграция с DeFi

Технологии

  • Microsoft SQL Server
  • .NET Framework
  • REST API
  • Web-интерфейс
  • Blockchain интеграция
