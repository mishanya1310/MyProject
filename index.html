Платформа для управления цифровыми активами

📊 О проекте

DAMP - это современная платформа для управления криптовалютными активами.

Система предоставляет инструменты для анализа, рекомендаций и управления портфелем цифровых активов.

💰 Пример данных криптовалют

Bitcoin (BTC)
$65,432.10
📈 +2.34% за 24ч
Ethereum (ETH)
$3,456.78
📈 +1.56% за 24ч
Cardano (ADA)
$0.5678
📉 -0.89% за 24ч

💡 Для реальных данных:

Запустите локальное API и откройте этот файл на своем компьютере

cd api
python app.py
затем откройте index.html в браузере

🎯 Функциональность

✅ Управление портфелями - Создание и мониторинг инвестиционных портфелей
✅ Рекомендательная система - Персонализированные предложения по активам
✅ Анализ рисков - Оценка риск-профиля и подбор активов
✅ REST API - Полнофункциональное API для работы с данными

🔧 Технологии

  • ✅ Python + Flask для API
  • ✅ HTML/CSS/JavaScript для интерфейса
  • ✅ CoinGecko API для реальных данных
  • ✅ GitHub Pages для хостинга
